# مواقيت الصلاة في مصر

واجهة تفاعلية تعرض أوقات الصلاة اليومية في مصر حسب المدينة المختارة.

## المميزات
- عرض أوقات الصلاة: الفجر، الظهر، العصر، المغرب، العشاء.
- عرض أوقات الشروق والغروب.
- عرض التاريخ الميلادي والهجري.
- تحويل الوقت من 24 ساعة إلى 12 ساعة بالعربية (صباحًا / مساءً).
- اختيار المدينة من قائمة جاهزة مع ترجمة اسم المدينة للغة العربية.
- تصميم متجاوب (Responsive) يدعم الشاشات الكبيرة والصغيرة.
- خلفية جميلة مع تأثير الـ blur للواجهة.

## التقنيات المستخدمة
- React (Functional Components + Hooks)
- Vite
- Tailwind CSS
- API: [Aladhan Prayer Times API](https://aladhan.com/prayer-times-api)
